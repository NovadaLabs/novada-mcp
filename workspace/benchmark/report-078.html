9. Methodology测试方法

  • Rounds: 1 warmup + 50 measured per (competitor × task).轮次：每组 1 次预热 + 50 轮测量。
  • Novada: Calls via MCP tool layer (build/ imports) — tests actual agent experience incl. parallel crawl, static fast path, dedup cache.Novada：通过 MCP 工具层调用——测试实际代理体验。
  • Competitors: Direct HTTP API calls.竞品：直接 HTTP API 调用。
  • Tavily: Pre-indexed (~120ms). Not comparable to live-fetch tools.Tavily：预建索引。与实时抓取不可直接对比。
  • BrightData: Excluded (no proxy zones).BrightData：已排除。
  • Environment: MacBook Pro, single location, sequential, 500ms delay.环境：MacBook Pro，单一位置，顺序执行，500ms 间隔。
