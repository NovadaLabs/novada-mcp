# 08 — Competitive Intelligence Analyst

## 1. Strongest Moat

**novada_research is unmatched.** No competitor offers a single-call multi-engine parallel search with auto-extraction and cited synthesis. Firecrawl requires agents to orchestrate search+scrape+synthesize manually. Tavily returns pre-indexed snippets, not live multi-source reports. This tool alone saves agents 5-10 tool calls per research task — a structural advantage that would take competitors months of pipeline engineering to replicate. The unified API key across search/proxy/browser/scrape further compounds switching costs.

## 2. Competitive Threat THIS MONTH

**Firecrawl's monitor system + JSON change tracking.** They shipped structured diff monitoring with goal-based judging — a sticky workflow that locks agents into their ecosystem. Once an agent sets up monitors, migration cost is high. BrightData's 69-tool sprawl with session_stats also threatens to become the "default enterprise MCP" before Novada reaches scale. Firecrawl's 494K downloads means ecosystem momentum — even with angry users, network effects compound.

## 3. Where to Attack

**Attack Firecrawl's pricing weakness, not Tavily.** Firecrawl's credit multipliers (5x for extract, 2x for search) create real user pain — $1/1K credits that actually cost $5/1K for common operations. Position novada_research as "one call, honest pricing" against their multi-call credit drain. Tavily's x402 micropayment experiment is niche and self-limiting. Firecrawl's 494K users with pricing frustration is the larger addressable market to convert.
